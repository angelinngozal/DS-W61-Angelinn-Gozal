
# Program Output

The program outputs: `"Meow"`
outputnya adalah Meow karena pada kelas Animal, dijelaskan bahwa output yang diinginkan adalah "some animal sound", namun setelah itu, terdapat kelas baru yaitu kelas Cat yang merupakan turunan alias inhertance dari Animal. pada kelas Cat ini, sound di-override atau diambil alih, memberi perintah untuk menghasilkan output "Meow"
java memilih untuk mengambil output dari Cat, karena saat menjalankan a.sound, java melihat ada variabel a, yang mana a ini merujuk kepada objek Cat. maka output yang dihasilkan pun sesuai dengan yang diinginkan objek Cat.
